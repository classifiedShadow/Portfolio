AI Arena

Project snapshot

AI Arena is a live competition platform designed for a college event environment where AI evaluation, human voting, and smooth round management all had to work together in real time.

The goal was to make the event feel polished for participants while keeping the judging flow fair and easy to run on a local network. That is why the system combines a web dashboard, AI scoring, leaderboard tracking, and round-based progression.

AI Arena started with a simple question: what happens when a competition platform is built so the model, the participants, and the audience all feel part of the same experience?

Round 1 — Theme-based image generation Participants generated images from a prompt theme. An Image Reward Model ranked submissions, while peer voting added a human layer and reduced friend-bias.
Round 2 — Prompt selection Participants chose the best prompt for an image. The options were intentionally close, so scoring rewarded nuance instead of obvious answers.
Round 3 — Similarity recreation Participants recreated a real image. AI similarity scored the results, top entries advanced, and the final judgment went to the chief guest.

AI Arena

Project snapshot

AI Arena is a live competition platform designed for a college event environment where AI evaluation, human voting, and smooth round management all had to work together in real time.

The goal was to make the event feel polished for participants while keeping the judging flow fair and easy to run on a local network. That is why the system combines a web dashboard, AI scoring, leaderboard tracking, and round-based progression.

AI Arena started with a simple question: what happens when a competition platform is built so the model, the participants, and the audience all feel part of the same experience?

Round 1 — Theme-based image generation Participants generated images from a prompt theme. An Image Reward Model ranked submissions, while peer voting added a human layer and reduced friend-bias.
Round 2 — Prompt selection Participants chose the best prompt for an image. The options were intentionally close, so scoring rewarded nuance instead of obvious answers.
Round 3 — Similarity recreation Participants recreated a real image. AI similarity scored the results, top entries advanced, and the final judgment went to the chief guest.
